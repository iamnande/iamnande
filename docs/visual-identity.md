# visual identity

## color system

| role       | value     |
|------------|-----------|
| background | `#101310` |
| surface    | `#161c16` |
| accent     | `#7a9e7e` |
| text       | `#dde8dd` |

## typography

| role                   | typeface            |
|------------------------|---------------------|
| ui chrome, labels, nav | geist mono          |
| headings               | geist mono          |
| article / body text    | atkinson hyperlegible |

## copy style

all copy is written in lowercase. be about yourself.

we don't need to fake such things with `text-transform`.

## motion

scrolled, smooth hovers.

movement must be derived from meaning.

it brings light without noise.

## density

pages should feel comfortable breathing softly, deeply.
